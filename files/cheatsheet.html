The six dplyr verbs

SPSS menuR verbDoes
Data > Select Casesfilter()Keep rows matching a condition
Variable Viewselect()Keep or drop columns
Transform > Computemutate()Create or change a column
Data > Sort Casesarrange()Sort rows
Data > Split Filegroup_by()Group before summarising
Data > Aggregatesummarise()Collapse groups to summaries

The pipe |> — read it as "and then"

visitors |>
  filter(origin == "United States") |>
  group_by(year) |>
  summarise(mean_spend = mean(avg_spending_usd))

Take visitors, and then keep US rows, and then group by year, and then compute mean spending. Each |> passes the result of the previous line into the first argument of the next.

Verbs in detail

# filter: rows
filter(visitors, origin == "United States", year == 2023)
# select: columns (use - to drop)
select(visitors, year, origin, visitors_stayover)
select(visitors, -satisfaction_score)
# mutate: new or recoded column
mutate(visitors,
       total = visitors_stayover + visitors_cruise,
       season = case_when(
         quarter %in% c(1,4) ~ "high",
         TRUE                ~ "low"))
# arrange: sort (desc for descending)
arrange(visitors, desc(visitors_stayover))
# summarise after group_by
visitors |> group_by(year) |>
  summarise(n = n(), mean = mean(avg_spending_usd))

Descriptive stats the SPSS way

# One variable at a time (Analyze > Descriptives)
mean(visitors$visitors_stayover, na.rm = TRUE)
median(x, na.rm = TRUE);  sd(x, na.rm = TRUE)
min(x);  max(x);  range(x)
quantile(x, c(.25, .5, .75), na.rm = TRUE)

# Frequency tables (Analyze > Frequencies / Crosstabs)
table(visitors$origin)
table(visitors$origin, visitors$year)

# Descriptives screen, summarise style
visitors |> group_by(origin) |> summarise(
  n    = n(),
  mean = mean(visitors_stayover, na.rm = TRUE),
  sd   = sd(visitors_stayover,   na.rm = TRUE))

Handling missing values

# How many NAs in a column?
sum(is.na(visitors$satisfaction_score))

# Drop rows with any NA, or just in chosen columns
drop_na(visitors)
visitors |> drop_na(satisfaction_score)

# Replace NA with a value
visitors |> replace_na(list(satisfaction_score = 0))

# Most stats functions need na.rm = TRUE
mean(visitors$avg_spending_usd, na.rm = TRUE)

SPSS marks missing in the Variable View; R stores NA directly in the column, and most functions refuse to compute through it unless you say so.

Visualisation, text analysis, troubleshooting

ggplot skeleton

ggplot(data = visitors,
       aes(x = year, y = visitors_stayover)) +
  geom_col() +
  labs(title = "Stayover visitors by year",
       x = "Year", y = "Visitors") +
  theme_minimal()

|> pipes data into a function. + adds layers to a ggplot. Do not mix them.

Common geoms

geom_col()Bar chart, values you supply
geom_bar()Bar chart, counts rows
geom_histogram()Distribution of one numeric
geom_point()Scatter plot
geom_line()Line chart, usually time
geom_boxplot()Distribution by group
geom_smooth()Trend line over points

Make a plot publication-ready

# Labels
labs(title    = "Stayover visitors",
     subtitle = "Aruba, 2019-2023",
     x = "Year", y = "Visitors",
     caption  = "Source: CBS Aruba",
     fill     = "Origin")

theme_minimal()  theme_classic()  theme_bw()
# DCDC-style colours
scale_fill_manual(values =
  c("#091f38", "#1a66b1", "#4a90c2"))
# Small multiples
facet_wrap(~ origin)
# Save to file
ggsave("figures/visitors.png",
       width = 7, height = 4, dpi = 300)

Text analysis and wordclouds

Tidy text analysis treats one word per row, so all dplyr verbs apply directly. Useful for free-text survey answers, transcripts, headlines, manifestos.

library(tidytext)
library(ggwordcloud)
# Start from a tibble with a text column, e.g. manifestos
word_counts <- manifestos |>
  unnest_tokens(word, text) |>          # one row per word
  anti_join(stop_words, by = "word") |> # drop the/and/...
  count(party_group, word, sort = TRUE)
# Top 50 words per group as a wordcloud
word_counts |> group_by(party_group) |> slice_max(n, n = 50) |>
  ggplot(aes(label = word, size = n, colour = party_group)) +
  geom_text_wordcloud() + scale_size_area(max_size = 14) +
  facet_wrap(~ party_group) + theme_minimal()

For Dutch or Papiamento, swap stop_words for a language-specific list or build your own tibble with a word column.

Joining two datasets

# left_join: keep all rows from x, add matching cols from y
visitors |> left_join(origin_codes, by = "origin")
# inner_join: rows in both; anti_join: x rows with no match in y

SPSS: Data > Merge Files > Add Variables, but less fragile.

When you get stuck

could not find function: you forgot library(...) at the top.
object ... not found: typo in an object or column name, or the line that creates it has not been run. Check the Environment pane.
unexpected ...: missing comma, bracket, or quote. Check the line above the error too.
Package missing: install.packages("name") once, then library(name) every session.
Help on any function: type ?function_name in the Console.
