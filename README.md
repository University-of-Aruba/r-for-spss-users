# Introduction to R for SPSS Users

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.XXXXXXX.svg)](https://doi.org/10.5281/zenodo.XXXXXXX)

A hands-on workshop that builds on what you already know from SPSS to get you
productive in R. No programming experience required.

This lesson is built with [The Carpentries Workbench][workbench] and developed
by the [DCDC Network](https://www.nwo.nl/en/projects/wcnjy25550) at the
[University of Aruba](https://www.ua.aw/).

## Course Overview

| | Session 1 | Session 2 |
|---|---|---|
| **Focus** | Why R, getting started, data manipulation, visualization | Statistical analysis, reproducible reporting, next steps |
| **Duration** | 5–6 hours (with breaks) | 5–6 hours (with breaks) |
| **Approach** | Every SPSS operation paired with its R equivalent | Hands-on exercises using Dutch Caribbean data |

## Episodes

1. **The Case for Switching** — live demo of what R can do that SPSS cannot
2. **Your First R Session** — RStudio orientation, importing data, descriptive statistics
3. **Data Manipulation** — filtering, sorting, recoding with tidyverse (mapped to SPSS menus)
4. **Visualization with ggplot2** — publication-quality charts
5. **Statistical Analysis** — t-tests, ANOVA, correlation, regression
6. **Reproducible Reporting** — R Markdown for combining analysis and write-up
7. **Where to Go from Here** — resources, Dutch Caribbean data packages, community

## Target Audience

Researchers, students, lecturers, and institutional staff who currently use SPSS
and want to transition to a free, open-source alternative. No programming
experience required — basic statistical knowledge is sufficient.
